# DacLB Test

This repository contains the completed tasks for the technical test

## Project Structure

- **[Ex1](./Ex1/)**: Implementation of Exercise 1 (Task details in its own [Readme](./Ex1/Readme.md)).
- **[Ex2](./Ex2/)**: Implementation of Exercise 2, involving OCR-based plan analysis, room detection, and wastewater pipe identification. (Detailed instructions in [Ex2/Readme](./Ex2/Readme.md)).

## Technical Summary

- **Core Technologies**: Python 3.10, OpenCV, EasyOCR, pdf2image (Poppler).
- **Key Algorithms**:
  - Multi-angle rotational OCR for label identification in CAD drawings.
  - Contour analysis and Non-Maximum Suppression (NMS) for spatial validation.
  - Morphological operations for feature extraction on architectural plans.
